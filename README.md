# 🎓 Web Dev Lab — Jornada Completa

> Plataforma de ensino interativa do HTML ao Laravel, com etapas progressivas, materiais de apoio e laboratórios práticos.

## 🗺️ Estrutura do Curso

| Etapa | Tecnologia | Nível |
|-------|-----------|-------|
| 1 | HTML5 Fundamentos | Iniciante |
| 2 | CSS3 & Design | Iniciante |
| 3 ⭐ | JavaScript Essencial | Intermediário *(Opcional)* |
| 4 | PHP Server-Side | Intermediário |
| 5 | Laravel: Ambiente & Laragon | Avançado |
| 6 | Laravel: Banco de Dados MySQL | Avançado |
| 7 | Laravel: Arquitetura MVC | Avançado |
| 8 | Laravel: Projeto Real (MVP) | Master |

## ⚙️ Tecnologias

- **React 19** + Vite 8
- **Tailwind CSS v4**
- **Lucide React** (ícones)
- Simulador PHP, SQL, Playground Frontend

## 🚀 Rodando Localmente

```bash
# Instalar dependências
npm install

# Servidor de desenvolvimento
npm run dev

# Build de produção
npm run build
```

## 📦 Deploy

Este projeto pode ser hospedado gratuitamente no **Vercel** ou **Netlify**.
